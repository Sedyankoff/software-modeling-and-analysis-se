## Описание на проекта
Проектът представлява опростена версия на Spotify – платформа за слушане на музика, управление на плейлисти, следване на артисти и използване на премиум абонаменти. Реализирана е релационна база данни с основните обекти на системата, Data Warehouse модел и Power BI отчет върху аналитичните данни. Проектът демонстрира пълния жизнен цикъл от ER модел → SQL база → DW → визуализации.

---

## Основни компоненти
- Концептуален модел (Chen)
- Логически модел (Crow’s Foot)
- MSSQL база данни (таблици, връзки, FK, функция, процедура, тригер)
- Напълнени примерни данни
- Data Warehouse диаграма (факт/измерения)
- Power BI отчет с 3 визуализации:
  - Top 10 най-слушани песни
  - Разпределение на слушанията по устройство
  - Месечни приходи от абонаменти
